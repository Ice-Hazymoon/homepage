# Ice-Hazymoon 的个人展示页

## demo

> [https://me.imiku.me](https://me.imiku.me)

## 运行

- 修改 `./src/mikuConfig.js` 配置文件
- `yarn` 或者 `npm install`
- `yarn serve` 或者 `npm run serve`

## 构建

`yarn build` 或 `npm run build`
